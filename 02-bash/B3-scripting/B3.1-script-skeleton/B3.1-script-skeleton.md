# KP B3.1 — Script Skeleton & Safe Header
**shebang · `set -e` `-u` `-o pipefail` · `IFS` · `usage()` · `$0`/`basename` · template**

**8 tasks · ⭐ · RHEL 10 · server1 · write each as a real script, `chmod +x`, run it**

> Scripts run against real systems, so "it runs" is not enough — it has to stop on error rather than pretend nothing happened. This section drills **the safety header every script should have**. The mantra: `#!/usr/bin/env bash` + `set -euo pipefail`.

---

**1.** Write `/root/bin/hi.sh` with a proper shebang that prints `ok`. Make it executable and run as `./hi.sh`.
*Acceptance:* prints `ok`; first line is `#!/usr/bin/env bash` (or `#!/bin/bash`).

**2.** Write a script with `set -e` that runs `false` and then an `echo`. Show the `echo` never runs (script aborts on the failed command).
*Acceptance:* nothing after `false` prints; exit status is non-zero.

**3.** Write a script with `set -u` that references an **unset** variable. Show it errors out instead of using an empty value.
*Acceptance:* prints `... : UNSETV: unbound variable` and exits non-zero.

**4.** Write a script with `set -eo pipefail` containing `false | true`. Show the script aborts (the hidden failure is caught).
*Acceptance:* the line after the pipeline never runs; exit non-zero.

**5.** Write `check.sh` with the full `set -euo pipefail` header and a `usage()` that prints `Usage: <name> <name>` to **stderr** and exits `1` when the argument count isn't exactly 1; otherwise print `Hello, <arg>`.
*Acceptance:* no args → usage on stderr, exit 1; one arg → `Hello, <arg>`, exit 0.

**6.** Write a script showing that `IFS` controls splitting: `IFS=: read -r a b c <<< "x:y:z"` then print `a|b|c`.
*Acceptance:* prints `x|y|z`.

**7.** Write a script that prints its own name using `$0` and `basename`.
*Acceptance:* run as `./t7.sh` → prints `script: t7.sh` (not the full path).

**8.** Write a complete starter template: shebang + `set -euo pipefail` + `IFS=$'\n\t'` + a `usage()` guard + a `main()` that echoes `processing <arg>`, called as `main "$@"`.
*Acceptance:* `./tpl.sh report.log` → `processing report.log`; no args → usage, exit 1.
