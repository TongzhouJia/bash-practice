# KP B3.6 Б─■ Arrays & Associative Arrays
**indexed arrays бЇ `${arr[@]}` `${#arr[@]}` `${!arr[@]}` бЇ `+=` бЇ slicing бЇ `mapfile` бЇ `declare -A` associative arrays бЇ counters**

**10 tasks бЇ Бґ░Бґ░ бЇ RHEL 10 бЇ server1 бЇ run each in bash and match the result**

> Samples are in `samples/`. Arrays hold a set of values; associative arrays (dictionaries) key on strings Б─■ **an associative array is the pure-bash version of an awk counter**, useful when you want grouped counts without shelling out to awk. Associative arrays must be declared with `declare -A` first.

---

**1.** Make an index array `fruits=(apple banana cherry)`; print the first element and the count.
*Acceptance:* `${fruits[0]}`Б├▓`apple`, `${#fruits[@]}`Б├▓`3`.

**2.** Append `date` with `+=`.
*Acceptance:* `fruits+=(date)` Б├▓ `apple banana cherry date`.

**3.** Iterate all elements safely (quoted `"${fruits[@]}"`).
*Acceptance:* prints each of the 4 elements.

**4.** Print all **indices** with `${!fruits[@]}`.
*Acceptance:* `0 1 2 3`.

**5.** Slice: elements 1..2 with `${fruits[@]:1:2}`.
*Acceptance:* `banana cherry`.

**6.** Load command output into an array with `mapfile -t` (read usernames from `passwd.sample`).
*Acceptance:* `mapfile -t users < <(cut -d: -f1 passwd.sample)` Б├▓ `count=10 first=root last=dave` (`${users[-1]}` is last).

**7.** Associative array: `declare -A color`, set `color[apple]=red` etc., read one back.
*Acceptance:* `${color[apple]}` Б├▓ `red`.

**8.** Iterate an associative array's keys and values (`${!color[@]}`).
*Acceptance:* (sorted) `apple=>red`, `banana=>yellow`.

**9.** Use an associative array as a **counter**: tally login shells (field 7) across `passwd.sample`.
*Acceptance:* (sorted) `/bin/bash 4`, `/bin/sh 1`, `/sbin/nologin 2`, `/usr/sbin/nologin 3`.

**10.** Split a string into an array on a delimiter with `read -ra`.
*Acceptance:* `IFS=, read -ra arr <<< "a,b,c"` Б├▓ `${arr[1]}`=`b`, count `3`.
