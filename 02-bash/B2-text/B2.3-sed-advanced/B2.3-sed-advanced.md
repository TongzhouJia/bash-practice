# KP B2.3 вҖ” sed Advanced
**multiple commands `-e`/`;` В· backreferences `\1` В· range addresses `/a/,/b/` В· `y///` В· address + substitute В· insert/append/delete `a/i/d`**

**8 tasks В· вӯҗвӯҗ В· RHEL 10 В· server1 В· run each against the sample files and match the result**

> Samples are in `samples/`. This section takes sed from a single substitution to structured rewriting: several steps in one pass, using `\1 \2` to capture and reorder matches, and selecting a span from X to Y.

---

**1.** In one `sed` pass over `sales.csv`, do **two** edits: drop the header row **and** turn commas into tabs.
*Acceptance:* `sed -e '1d' -e 's/,/\t/g'` вҶ’ first line `2026-07-01вҮҘEastвҮҘWidgetвҮҘ10вҮҘ2.50` (real tabs).

**2.** Reformat each `passwd.sample` line to `user -> shell` using capture groups.
*Acceptance:* `sed -E 's/^([^:]+):.*:([^:]+)$/\1 -> \2/'` вҶ’ `root -> /bin/bash`, `daemon -> /usr/sbin/nologin`, вҖҰ

**3.** Reorder a date `2026-07-01` into `01/07/2026` with three capture groups.
*Acceptance:* `sed -E 's#([0-9]+)-([0-9]+)-([0-9]+)#\3/\2/\1#'` вҶ’ `01/07/2026`.

**4.** Print the block of `access.log` lines **from** the first `11:00:00` **to** the first `11:03:00` (range between two patterns).
*Acceptance:* `sed -n '/11:00:00/,/11:03:00/p'` вҶ’ those 4 consecutive lines.

**5.** Transliterate `abc` вҶ’ `ABC` using `y///` (sed's version of `tr`).
*Acceptance:* `echo "abc" | sed 'y/abc/ABC/'` вҶ’ `ABC`.

**6.** In `sales.csv`, uppercase `Widget` вҶ’ `WIDGET` **only on lines containing `East`** (leave other regions' Widget untouched).
*Acceptance:* `sed '/East/ s/Widget/WIDGET/'` вҶ’ East rows show `WIDGET`; North/West rows still show `Widget`.

**7.** Strip all blank lines from the stream `printf 'a\n\nb\n\n\nc\n'`.
*Acceptance:* `sed '/^$/d'` вҶ’ `a`, `b`, `c` (no empties).

**8.** Append a new line `INSERTED` **after line 2** of the stream `printf 'a\nb\nc\n'`.
*Acceptance:* `sed '2a INSERTED'` вҶ’ `a`, `b`, `INSERTED`, `c`.
