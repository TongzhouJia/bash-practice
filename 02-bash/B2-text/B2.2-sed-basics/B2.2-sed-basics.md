# KP B2.2 — sed Basics
**`s/old/new/` · `/g` `/2` `/i` · addresses (`N` `N,M` `$` `/pat/`) · `-n p` · `d` · `-i` · alternate delimiters · `&`**

**10 tasks · ⭐⭐ · RHEL 10 · server1 · run each against the sample files and match the result**

> Samples are in `samples/` (`cd` in first). sed is a **stream editor**, working line by line. The core action is a single `s/find/replace/`, and addresses decide which lines it touches. ⚠️ `-i` **really modifies the file** — always `cp` a copy and edit that while practicing.

---

**1.** In `passwd.sample`, substitute `/bin/bash` → `/bin/zsh` (first match per line) and count how many lines changed.
*Acceptance:* `sed 's#/bin/bash#/bin/zsh#' … | grep -c /bin/zsh` → `4`.

**2.** Turn every comma in `sales.csv` into a pipe `|` (all occurrences per line).
*Acceptance:* `sed 's/,/|/g'` → `date|region|product|units|price`, `2026-07-01|East|Widget|10|2.50`, …

**3.** In the string `1 2 3 4`, replace **only the 2nd** space with `_`.
*Acceptance:* `echo "1 2 3 4" | sed 's/ /_/2'` → `1 2_3 4`.

**4.** Print **only lines 2–4** of `passwd.sample` (suppress the rest).
*Acceptance:* `sed -n '2,4p'` → the `daemon`, `bin`, `tongzhou` lines.

**5.** Print `sales.csv` **without its header row** (delete line 1).
*Acceptance:* `sed '1d'` → output starts at `2026-07-01,East,Widget,10,2.50`.

**6.** Delete every line matching `nologin` from `passwd.sample`; how many lines remain?
*Acceptance:* `sed '/nologin/d' … | wc -l` → `5` (only the real login accounts remain).

**7.** Print just the **last line** of `passwd.sample` using the `$` address.
*Acceptance:* `sed -n '$p'` → the `dave:…` line.

**8.** On a **copy** of `passwd.sample`, do an in-place rename of `tongzhou` → `tzhou` **while keeping a backup**.
*Acceptance:* `cp passwd.sample /tmp/p && sed -i.bak 's/tongzhou/tzhou/g' /tmp/p` — `/tmp/p` is changed and `/tmp/p.bak` holds the original.

**9.** In `passwd.sample`, change the home-directory prefix `/home` → `/users`, using a **different delimiter** so you don't have to escape the slashes.
*Acceptance:* `sed 's#/home#/users#g'` → `…:/users/tongzhou:…`, `…:/users/alice:…`, …

**10.** Wrap the matched text in brackets using `&` (the whole match): turn `error 500` into `error [500]`.
*Acceptance:* `echo "error 500" | sed 's/500/[&]/'` → `error [500]`.
