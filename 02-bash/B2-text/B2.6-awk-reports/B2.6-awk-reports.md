# KP B2.6 — awk Associative Arrays & Reports ⭐⭐⭐
**`count[$k]++` · `for(k in arr)` · top-N · grouped sums · distinct counts · compound keys · reports**

**12 tasks · ⭐⭐⭐ · RHEL 10 · server1 · run each against the sample files and match the result**

> Samples are in `samples/`. **This is the most useful section in the whole set.** An associative array is a dictionary keyed by arbitrary strings; `count[$1]++` does "group and count by a column" in one line. Log analysis, top-N and grouped summaries all ride on it. Slow down and get every task properly.

---

**1.** Count requests **per IP** in `access.log`.
*Acceptance:* `awk '{c[$1]++} END{for(ip in c) print c[ip], ip}' | sort -rn` → `10 10.0.0.1`, `5 10.0.0.2`, `4 …`.

**2.** Show the **top 3** IPs by request count.
*Acceptance:* pipe task 1 through `sort -rn | head -3` → `10.0.0.1`(10), `10.0.0.2`(5), `203.0.113.7`(4).

**3.** Count requests **per status code** (sorted).
*Acceptance:* `awk '{c[$9]++} END{for(s in c) print s, c[s]}' | sort` → `200 17`, `301 2`, `401 1`, `404 5`, `500 3`, `503 2`.

**4.** Total **revenue by region** from `sales.csv` (Σ units×price), 2 decimals.
*Acceptance:* `awk -F, 'NR>1{r[$2]+=$4*$5} END{for(x in r) printf "%s %.2f\n",x,r[x]}' | sort` → `East 196.96`, `North 191.41`, `West 117.94`.

**5.** Total **units sold per product**.
*Acceptance:* `awk -F, 'NR>1{u[$3]+=$4} END{for(p in u) print p,u[p]}' | sort` → `Gadget 19`, `Gizmo 16`, `Widget 37`.

**6.** How many **distinct IPs** appear in `access.log`?
*Acceptance:* `awk '{seen[$1]++} END{print length(seen)}'` → `6`.

**7.** Count **failed SSH logins per source IP** in `syslog.log` (the IP is the token after `from`).
*Acceptance:* `awk '/Failed password/{for(i=1;i<=NF;i++) if($i=="from") c[$(i+1)]++} END{for(x in c) print c[x],x}' | sort -rn` → `3 203.0.113.7`, `1 198.51.100.9`.

**8.** Count rows by the **composite key** region+product in `sales.csv`.
*Acceptance:* `awk -F, 'NR>1{c[$2","$3]++} END{for(k in c) print k,c[k]}' | sort` → `East,Gadget 1`, `East,Gizmo 2`, `East,Widget 2`, …

**9.** Total **bytes served per IP** (sum of last field), ranked.
*Acceptance:* `awk '{b[$1]+=$NF} END{for(ip in b) print b[ip],ip}' | sort -rn` → top is `8256 10.0.0.2`, then `5376 10.0.0.1`, …

**10.** Print the **single most common** status code and its count (find the max in `END`).
*Acceptance:* `awk '{c[$9]++} END{for(s in c) if(c[s]>m){m=c[s];top=s}; print top,m}'` → `200 17`.

**11.** Count requests **per hour** (hour = 2nd colon-field of the timestamp `$4`).
*Acceptance:* `awk '{split($4,a,":"); c[a[2]]++} END{for(h in c) print h,c[h]}' | sort` → `10 14`, `11 16`.

**12.** Capstone report: for each status code print `code count percentage` of total.
*Acceptance:* `awk '{c[$9]++; t++} END{for(s in c) printf "%s  %2d  %.1f%%\n",s,c[s],100*c[s]/t}' | sort` → `200  17  56.7%`, `404   5  16.7%`, `500   3  10.0%`, …
