# KP B4.1 — Log Analysis ⭐⭐⭐
**combining grep/awk/sort for real log analysis: counts, top-N, error rates, time windows, security events, report scripts**

**12 tasks · ⭐⭐⭐ · RHEL 10 · server1 · run against the sample logs and match the result**

> Samples are in `samples/` (`access.log`, 30 lines, and `syslog.log`). **This is where the B2 techniques meet a real scenario** — log analysis is the number one use for Bash on a live system. Write it yourself first, then check.

---

**1.** Total requests and number of **unique** visitor IPs in `access.log`.
*Acceptance:* `total=30`, `unique=6`.

**2.** Top 5 IPs by request count.
*Acceptance:* `10.0.0.1`(10), `10.0.0.2`(5), `203.0.113.7`(4), `192.168.1.50`(4), `10.0.0.3`(4).

**3.** Count of each HTTP status code.
*Acceptance:* `200`×17, `404`×5, `500`×3, `503`×2, `301`×2, `401`×1.

**4.** **Error rate**: percentage of requests with status ≥ 400.
*Acceptance:* `awk '$9>=400{e++} END{printf "%.1f%%\n",100*e/NR}'` → `36.7%` (11/30).

**5.** **Top 3 IPs causing 5xx errors** (the classic "who's breaking prod").
*Acceptance:* `awk '$9~/^5/{print $1}' | sort | uniq -c | sort -rn | head -3` → `10.0.0.1`(4), `203.0.113.7`(1).

**6.** Total bytes served and average response size.
*Acceptance:* total `23936`; average `797.9`.

**7.** Requests **per hour** (traffic over time).
*Acceptance:* `10`→14, `11`→16.

**8.** The single **most-requested URL path** (field 7).
*Acceptance:* `/index.html` (7 hits); next `/api/data` (5), `/products` (4).

**9.** From `syslog.log`, **failed SSH logins per source IP** (security triage).
*Acceptance:* `203.0.113.7`×3, `198.51.100.9`×1.

**10.** Count requests in the **time window** 11:00–11:05.
*Acceptance:* `awk '$4 ~ /:11:0[0-5]:/' access.log | wc -l` → `6`.

**11.** The **busiest hour** (peak).
*Acceptance:* hour `11` with `16` requests.

**12.** Write `weblog-report.sh <logfile>` that prints a summary: total requests, unique IPs, top IP, error count, top status. Run it on `access.log`.
*Acceptance:* prints `Total requests: 30`, `Unique IPs: 6`, `Top IP: 10.0.0.1 (10)`, `Error requests: 11`, `Top status: 200 (17)`.
